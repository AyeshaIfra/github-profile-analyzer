# GitHub Profile Analyzer API

## Tech Stack
- Node.js
- Express.js
- MySQL
- GitHub API

## Live API
https://your-app.vercel.app/

## Endpoints

### Analyze GitHub user
GET /api/analyze/:username

### Get all profiles
GET /api/profiles

### Get single profile
GET /api/profiles/:id

## Setup Locally

npm install
npm run dev
